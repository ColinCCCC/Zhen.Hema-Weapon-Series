## 成品说明

本项目为用于兵击对练的武器系列设计，包含腰刀、打刀、单手德砍。系列设计主题风格为激进、锐利的近未来工业风格。

刀身无渐薄，项目通过直接设计刀身形状，以符合真刀质量分布，并达成减伤要求。
